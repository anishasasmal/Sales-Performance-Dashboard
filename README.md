# Sales-Performance-Dashboard
Interactive Sales Performance Dashboard built using Power BI to analyze sales, profit, cost, and order trends across products, cities, countries, and time periods. Features data modeling, DAX, Power Query, KPI cards, and interactive visualizations to deliver actionable business insights.

📊 Salesable Marketing Solutions - Sales Performance Dashboard

📌 Project Overview :-
This project presents an interactive Sales Performance Dashboard developed in Power BI to analyze key business metrics such as sales, profit, cost, and order quantity. The dashboard enables users to explore business performance across different years, cities, countries, months, and product categories through interactive visualizations.

🎯 Project Objective :-
-Monitor overall sales performance.
-Analyze profit and cost trends.
-Identify top-performing products and regions.
-Compare sales across cities and countries.
-Provide actionable insights for better business decisions.

📊 Dashboard KPIs :-
💰 Total Sales: 55.16M
📈 Total Profit: 31.49M
💵 Total Cost: 23.67M
📦 Total Order Quantity: 251K

📌 Dashboard Features:-
-Interactive slicers for Year/Month/City/Product Name
-KPI Cards
-Sales vs Profit Trend Analysis
-Product Category Sales Distribution
-Region-wise Sales Analysis
-Country & City-wise Profit Analysis
-City-wise Sales Comparison
-Product-wise Sales, Cost & Profit Summary

🛠 Tools & Technologies:-
-Power BI
-Power Query
-DAX
-Data Modeling
-Data Cleaning
-Data Visualization

📈 Key Insights:-
-Total sales reached 55.16M during the selected period.
-Computers generated the highest sales among all product categories.
-Sales and profit showed significant growth in 2024.
-Beijing recorded the highest city-wise sales.
-Interactive filters allow users to explore trends by year, month, city, and product.
